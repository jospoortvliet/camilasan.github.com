RVM

RVM is a command-line tool which allows you to easily install, manage, and work with multiple ruby environments from interpreters to sets of gems.

More info at http://rvm.io

top

Setting up your dev environment

Linux

  • Install RVM with Ruby and Rails: $ \curl -sSL https://get.rvm.io | bash -s stable --rails
  • To check your installation run: $ rvm --version
    $ ruby --version
    $ rails --version
  • For coding you can use any text editor you like. I suggest Sublime Text: http://www.sublimetext.com/2. It has a good support for Ruby and Rails.

More info at http://rvm.io

OS X

  • You can see the howto here: http://guides.railsgirls.com/install/#setup-for-os-x

Windows

  • You can see the howto here: http://guides.railsgirls.com/install/#setup-for-windows
  • Just remember that on Windows you need to use the command line terminal that comes with your Rails installation. The Windows cmd doesn't load rvm.
